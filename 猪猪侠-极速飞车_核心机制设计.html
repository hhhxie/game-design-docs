🔀 漂移判定流程

转向角度 > 45° ──否──→ 正常转向 │ 是 │ 持续 > 0.5s? ──否──→ 正常转向 │ 是 ↓ 【进入漂移】 │ ┌────┴────┐ │ 持续漂移 │ → 积累氮气 + 技能能量 │ 速度×80% │ └────┬────┘ │ 回正手柄？──否──→ 继续漂移（无时间上限） （角度 < 10°） │ 是 ↓ 【退出漂移】 │ ↓ 【回正小加速】←── +20%速度，持续2秒，不消耗氮气 │ ↓ 【加速后逐渐降速】←── 不是瞬间回到常规速度，而是平缓衰减

📈 氮气积累与消耗

短漂移 <1s ~25% 充能 → 按住扳机约0.25秒
中漂移 1~3s ~50-75% 充能 → 按住扳机约0.5-0.75秒
长漂移 3s以上 接近满格 → 按住扳机约0.75-1秒
⭐ 氮气从0充至满 = 完美漂移 → 额外注入1格氮气（+1秒释放时间）
✅ 按住扳机键 → 氮气持续消耗（1格/秒，0.1格/0.1秒）+ 释放加速（+30%）| 松开扳机键 → 立即停止，保留剩余氮气

🔬 氮气消耗精度

氮气消耗以0.1秒为最小单位进行精确计算，让玩家灵活控制用量

按住时间消耗氮气典型场景
0.1秒0.1格最小消耗单位，微加速调整
0.3秒0.3格短按微调速度
0.5秒0.5格半秒点按
1.0秒1.0格标准消耗
2.0秒2.0格中等释放
3.0秒3.0格长时间冲刺

📉 加速后降速衰减

氮气加速结束后，速度不会瞬间降回常规最高速，而是经过一段降速衰减期平缓回归，给玩家更多时间适应速度变化。

参数默认值说明
衰减方式线性递减从加速最高速度平滑下降至常规最高速
衰减时长加速持续 ×50%加速3秒则衰减1.5秒
衰减中转向-5%恢复正常转向衰减
衰减中碰障碍正常惩罚不享受加速状态特殊处理

🎬 表现形式

视觉回正小加速：车尾短暂蓝色闪光 + 轻微速度线 | 氮气加速：车尾蓝紫色氮气火焰 + 两侧速度线 + 画面边缘动态模糊 + 松开扳机键时氮气火焰瞬间消失 + 衰减期：火焰逐渐缩小、速度线变淡、模糊缓缓消退
音效回正小加速：短促"嗖"声 | 氮气释放喷射"嗖——"声 + 加速中引擎更高亢 + 衰减期：引擎声逐渐降回正常
UI回正小加速："漂移加速！"2秒提示 | 氮气加速：氮气条按0.1格精度实时减少 + 时速表攀升后缓缓回落 + 加速期"氮气加速！" → 衰减期"降速中..."

🚧 阻碍机关

赛道中的障碍物，碰撞后造成减速或失控（障碍物仅与赛道设计有关，不受难度影响）

🔶 路障

速度 -20%，持续2秒
视觉：黄黑条纹三角锥/路障桩
反馈：赛车弹开微幅偏移 + 碰撞火花
音效：撞击声 + 塑料碰撞声
频率：全关卡均出现，初期主要障碍

🖤 油污

赛车打滑失控，左右摇晃2秒
视觉：赛道地面黑色油渍（反光效果）
反馈：赛车左右摇摆 + 轮胎打滑痕迹
音效：轮胎打滑"吱吱"声
频率：中后期关卡开始出现

🔲 窄道

碰触墙壁速度 -30%，持续1.5秒
视觉：赛道收窄区域，护栏/岩壁
反馈：碰撞墙壁弹回 + 擦痕特效
音效：金属刮擦声
频率：全关卡出现，后期收窄更剧烈

📏 障碍物密度 & 氮气碰撞

ℹ️ 障碍物密度仅与关卡赛道设计相关，不受难度选择影响
关卡每百米障碍数主要类型
第1-2关1~2个路障为主
第3-4关2~3个路障 + 窄道
第5-6关3~4个路障 + 窄道 + 油污
第7-8关4~5个全类型混合
